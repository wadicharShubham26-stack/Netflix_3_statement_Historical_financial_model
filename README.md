# Netflix_3_statement_Historical_financial_model
The  project objective is to understand the financial performance of Netflix from 2023 to 2025.

This project contains an integrated 3 statement historical financial analysis model. 
It includes key measures and ratios with interpretations. 
The data was extracted from the Netflix 10 k report and an indirect summary method is used.
The tool used was Microsoft Excel.

The final summary of financial analysis reflects that Netflix had strong revenue growth.
The margins indicate strenthenging profitability and operational efficiency.
